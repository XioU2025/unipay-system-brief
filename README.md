# UniPay – Sistema de Pago de Colegiaturas en Línea

## Descripción

UniPay es un sistema web diseñado para gestionar pagos de colegiatura universitaria de forma digital, permitiendo registrar estudiantes, consultar saldos y realizar pagos con actualización automática.

## Problema

Los procesos manuales de pago generan errores en registros, retrasos en actualización de saldos y falta de trazabilidad financiera.

## Alcance

- Registro de estudiantes
- Consulta de saldo
- Pago mensual
- Actualización automática del saldo
- Historial de pagos

## Fuera de alcance

- Integración bancaria real
- Facturación electrónica oficial
- Gestión académica
